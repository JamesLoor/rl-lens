Activá la Stats API

RL Insights necesita que Rocket League tenga la Stats API local habilitada.

1
Encontrá el archivo de config ~/Library/Application Support/Rocket League/TAGame/Config/DefaultStatsAPI.ini %APPDATA%\Rocket League\TAGame\Config\DefaultStatsAPI.ini
2
Añadí estas líneas 
[LanStatsServer]
bEnabled=True
WebSocketEnabled=True
Port=49122
3
Reiniciá Rocket League
